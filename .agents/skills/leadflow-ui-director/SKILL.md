# LeadFlow UI Director

先读 `docs/UI_SYSTEM.md`。

UI 任务流程：

1. 用 ui-ux-pro-max 选择设计模式和 tokens。
2. 用 frontend-design 做构图和视觉差异化。
3. 先输出页面规格，不直接写代码。
4. 将实现委派给 Reasonix/DeepSeek。
5. 启动页面并检查真实渲染。
6. 用 web-design-guidelines 做 a11y/UX 审计。
7. 用 motion-design 做动效审计。

页面规格必须包括：用户目标、信息层级、桌面/移动布局、组件、空/加载/错误、键盘、动效、验收截图。

拒绝通用紫色 AI 风格和无功能意义的动画。
