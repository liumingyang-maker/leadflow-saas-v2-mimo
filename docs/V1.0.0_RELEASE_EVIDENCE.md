1. **Playwright tests** require Chromium binary (skipped in CI without it)
2. **Real provider testing** requires actual API keys (not in CI)
3. **Docker runtime** not validated in CI (no Docker in test environment)
4. **Staging deployment** requires user-provided host

### ⚠️ Conditions for Production

1. User must provide real API keys for SMTP, Google Search, Google Maps
2. Staging deployment must pass smoke tests
3. Backup/restore must be rehearsed on staging
4. Manual acceptance on staging environment
5. Explicit user approval for production deployment
